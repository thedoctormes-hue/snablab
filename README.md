## 🎯 Что это?

**SnabLab** — это SaaS-платформа для автоматизации закупок лабораторных расходных материалов. Создана на основе реального опыта работы клинико-диагностической лаборатории.

### Проблема

Лаборатории тратят **десятки часов** в месяц на:
- Ручной разбор коммерческих предложений (PDF, DOCX)
- Сопоставление позиций с внутренней номенклатурой
- Составление служебных записок и заявок
- Контроль сроков годности реагентов
- Поиск выгодных предложений на рынке

### Решение

SnabLab автоматизирует **весь цикл закупок**:

```
📄 КП (PDF/DOCX) → 🤖 Парсинг → 📊 Сопоставление → 📝 СЗ → ✅ Заявка → 📦 Склад
```

## ✨ Ключевые возможности

### 📄 Интеллектуальный парсинг коммерческих предложений
- Автоматический разбор PDF и DOCX от поставщиков
- Поддержка форматов: КардиоМед, ИМБИАН ЛАБ и других
- Извлечение цен, артикулов, сроков поставки

### 🤖 Автосопоставление позиций
- Автоматическое сопоставление позиций КП с внутренней номенклатурой
- Нечёткое сопоставление с учётом синонимов и вариантов написания
- Контроль расхождений и дубликатов

### 📊 Ценовой анализ
- Сравнение цен между поставщиками
- Анализ маржинальности
- История цен по каждой позиции
- Выявление аномальных цен

### 🏛 Мониторинг госзакупок
- Автоматический парсинг zakupki.gov.ru
- Поиск релевантных закупок по номенклатуре
- Анализ цен госконтрактов
- Уведомления о новых закупках

### 📦 Складской учёт
- Учёт партий с контролем сроков годности
- Автоматические алерты при низком остатке
- Учёт годного/бракованного/истёкшего товара
- Инвентаризация

### 📝 Генерация документов
- Автоматическая генерация служебных записок (DOCX)
- Формирование заявок на закупку
- Экспорт данных в CSV

### 🤖 Telegram-бот
- Мониторинг закупок в реальном времени
- Алерты о низких остатках
- Уведомления о новых госзакупках
- Быстрый доступ к аналитике

## 🚀 Быстрый старт

### Docker Compose

```bash
git clone https://github.com/thedoctormes-hue/snablab.git
cd snablab
cp .env.example .env
docker compose up -d

# Frontend: http://localhost:5173
# API Docs: http://localhost:8000/docs
```

### Локальная разработка

```bash
# Backend
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
alembic upgrade head
uvicorn app.main:app --reload --port 8000

# Frontend
cd frontend
npm install
npm run dev
```
