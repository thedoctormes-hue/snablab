# 🗺️ СнабЛаб — Roadmap развития

> **Проект Бестии** | Владелец: Бестия (лаборант-силовик)
> Обновлено: 2026-05-15

## Текущее состояние: v0.1.0-alpha

FastAPI backend с полной моделью данных, парсером КП и генератором СЗ. Frontend не реализован.

### Что готово
- ✅ Модель данных (8 сущностей, полные связи)
- ✅ CRUD API для всех доменов (7 модулей)
- ✅ Парсер PDF/DOCX коммерческих предложений
- ✅ Генератор DOCX служебных записок
- ✅ JWT аутентификация (login/register)
- ✅ Alembic миграции
- ✅ Docker Compose (PostgreSQL, Redis, MinIO, backend, frontend)
- ✅ Seed-скрипт справочников
- ✅ Тесты (6/6 пройдено)
